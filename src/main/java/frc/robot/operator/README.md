# Operator Controls

| Input             | Description                               |
|-------------------|-------------------------------------------|
| cross             | Prime Speaker                             |
| R1 + cross        | Shoot Speaker                             |
| square            | Prime Amp                                 |
| R1 + square       | Shoot Amp                                 |
| R2 + Right Y Axis | Elevator Control (Manual)                 |
| L2 + Left X Axis  | Intake Pivot Control (Manual - Override)  |
| L2 + Right X Axis | Shooter Pivot Control (Manual - Override) |
